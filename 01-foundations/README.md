# 📘 Phase 1 — Foundations of Agentic AI

> **Goal:** Build a rock-solid mental model of what agents are, how they work, and what makes them fundamentally different from traditional software — before we talk about securing them.

---

## Why Start Here?

You can't secure something you don't understand. Every security concept in this repo — prompt injection, memory poisoning, MCP attacks — only makes sense once you know *what agents are made of*.

These 9 articles build that foundation.
